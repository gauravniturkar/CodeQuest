#  CodeQuest — Learn to Code, Gamified

> An AI-powered, gamified coding education platform. Learn Python concepts through quizzes, challenges, and a global leaderboard — with fresh questions generated by Claude AI every time.

![CodeQuest](https://img.shields.io/badge/built%20with-Claude%20AI-orange) ![License](https://img.shields.io/badge/license-MIT-blue)

##  Features

- **7 Learning Sections** — Variables, Functions, Loops, Conditionals, Lists, OOP, APIs
- **AI-Generated Questions** — Claude generates fresh, unique questions every session
- **Global Leaderboard** — compete with learners worldwide (powered by artifact storage)
- **XP & Streak System** — earn XP, maintain streaks, unlock sections
- **Hint System** — AI-powered contextual hints
- **Fully Static** — no backend needed, deploy anywhere

##  AI Questions Setup

The game uses the Claude API to generate questions. The API key is handled by the Claude.ai artifact environment automatically — no setup needed when running through Claude.ai.

For standalone deployment, the game gracefully falls back to a built-in question bank when the API is unavailable.

##  Sections Covered

| # | Topic | Concepts |
|---|-------|----------|
| 1 | Variables | Types, assignment, naming |
| 2 | Functions | def, return, parameters |
| 3 | Loops | for, while, break, continue |
| 4 | Conditionals | if/elif/else, comparison ops |
| 5 | Lists | indexing, slicing, methods |
| 6 | OOP | classes, __init__, methods, inheritance |
| 7 | APIs | requests, JSON, status codes, REST |

##  Tech Stack

- Pure HTML/CSS/JS — zero dependencies
- Claude Sonnet API — AI question generation
- Artifact persistent storage — leaderboard
